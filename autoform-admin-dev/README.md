## Installation

Follow these steps to install and set up the project locally:

1. Clone the repository:

   ```bash
   git clone https://github.com/sandeepgoyalsoftmine/autoform-admin.git

   ```

2. Navigate to the project directory:

   cd autoform-admin

3. Install the dependencies:

   npm install

   Make sure you have Node.js version 14.17.0 installed. You can check the installed version by running node -v in your terminal.
